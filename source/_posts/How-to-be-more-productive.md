> 看了快20年的电视剧和电影，往往我都在想，镜头的一下秒会发生什么，导演为何要如此安排，如果换一种表达方式效果会不会好点。是的，这也是成为了我们经常吐槽的地方，可归纳如下几个原因：本身条件限制、外界条件限制……换言之，我们应该如何高效提高自己的专业技能，提高做事的效率，简化复杂的问题，这些需要我们专业技能过足，有夯实的经验积累。


{% note default %}  

How to be more productive

{% endnote %}


{% note primary %}  

- 选择一个好问题

- 学会归纳总结问题

- 列个清单、掌握大局

- 思考清楚、画出流程

- 化整为零、各个击破

- 放松自己、缓解压力

{% endnote %}
